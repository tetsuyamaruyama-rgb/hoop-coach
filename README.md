# バスケコーチ Kids 🏀

7歳向け・バスケの動きをスマホで撮って **その場でAI分析** する PWA（インストール型Webアプリ）。
動画は端末から出ません（ブラウザ内でポーズ解析）。初回読込後は **完全オフライン** で動きます。

## しくみ
- **ポーズ解析**: TensorFlow.js + MoveNet（端末内・WebGL）
- **アドバイス**: ルールベース（オフライン・無料）。7歳向けに「ほめ1つ＋なおす1つ」だけ
- **3つのモード**:
  - 🎯 **シュート**: ひざのまげ / うでの伸び / フォロースルー / あしのはば
  - ⛹️ **ドリブル**: ひくいしせい / あたまアップ（ボールを見ない） / リズム（手の動き）
  - 🛡️ **ディフェンス**: こしを おとす / あしのはば / てを あげる
- **れんしゅうの きろく**: 各回の⭐を端末内（localStorage）に保存。1週間の練習回数を棒グラフで表示。データは端末から出ません。

## ローカルで動かす（PC確認用）
```
cd C:\dev\hoop_coach
python -m http.server 8731 --directory C:\dev\hoop_coach
```
→ ブラウザで http://127.0.0.1:8731

※ ファイル選択（動画アップロード）方式なので、PCでも動画ファイルがあれば試せます。

## iPhoneに「アプリ」として入れる手順（HTTPSが必要）
iOSはカメラ／PWAインストールに **HTTPS** が必須。無料でHTTPS配信する一番かんたんな方法：

### 方法A: Netlify Drop（最速・アカウント任意）
1. https://app.netlify.com/drop を開く
2. `C:\dev\hoop_coach` フォルダを **ドラッグ&ドロップ**
3. 発行された `https://◯◯.netlify.app` を **iPhoneのSafari** で開く
4. 共有ボタン → **「ホーム画面に追加」** → アプリアイコンができる
5. 一度開けばモデルがキャッシュされ、以降は **オフライン**でも動く

### 方法B: GitHub Pages（無料・URL固定）
1. リポジトリを作って中身をpush
2. Settings → Pages → ブランチ公開
3. `https://<user>.github.io/<repo>/` をiPhoneで開く → ホーム画面に追加

## 使い方（親子で）
1. ふだんのカメラアプリで **横から** シュートを3〜5秒録画
2. アプリを開く →「どうがを えらぶ」→ その動画を選ぶ
3. ⭐スコア＋アドバイス＋骨格をなぞった静止画が出る
